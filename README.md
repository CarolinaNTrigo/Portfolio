## Publicação no GitHub Pages

Para publicar:

1. Crie um repositório público.
2. Envie os arquivos do projeto.
3. Vá em **Settings > Pages**.
4. Escolha a branch de publicação.
5. Salve e aguarde a geração da URL.

Se quiser usar este repositório como perfil público, o nome do repositório deve ser igual ao seu nome de usuário no GitHub, e o arquivo `README.md` precisa estar na raiz. [web:116]
